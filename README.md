你好，在下洛樱，你也可以叫我天才少女洛樱。

今年3月份我与ChatGPT交流，他问我：
> 如果回到大一，你看到一个和你现在的情况一样的学长，你会怎么想？

这个问题着实让我一怔。我在本科没有发出任何论文，也没有一个拿得出手的竞赛奖项，更没有亮眼的人际关系，确实是相当失败的本科四年；但是如果在当时那个刚刚毕业的高中生看来，现在的我想必能给18岁的自己交出一份满意的答卷。

我学到了之前初高中时期没有机会接触到的各种技术，掌握了一定的沟通能力，更重要的是，通过这四年的经历，我明白了过往中的自己最大的一个缺陷，那就是把做题考试的成绩看的太重。

初高中是一个一切向成绩看齐的时代。尤其在河北的中学里，一切对提高成绩没有直接、显著意义的行为，都是大逆不道的，来自身后的目光随时会打击到每一个学生头上。固然在经济不发达的地区，这样的高压环境确实提高了很多人的成绩，但它永远无法解答这个问题：高考后，我们做什么？

是啊，做什么？

来自全省各市的16岁左右的少年聚集到一个个教室里，教室里的空气混合着一周，甚至一个月没洗澡的体臭和头油的气味，让进班讲课的女老师面露难色。盛夏的夜晚，是两台16度的立式空调仍然难以排解的炎热，背上的汗水与潮湿的校服相互黏合，提醒着蠕动的身体，这里并不是人类宜居的环境。你每天早上六点被宿管的哨声和起床铃声叫醒，顾不上全身肌肉的酸痛便冲出宿舍楼，在清晨的阳光或者漆黑的夜空下跑操——这倒不是因为你多爱运动，而是无数个眼睛盯着你的一举一动，让你的每一秒都被安排的明明白白。就这样循环往复三年，高考最后一门的收卷铃响起，答题卡被监考老师收走。在天国般的宁静和幸福中，自由的空气无声的向你宣告：你自由了。

你如愿进入理想的大学，却发现这里的一切是那样的陌生。五湖四海的同学们似乎都身怀绝技，高中对他们来说不只是一个做题培训班、一个监狱。没有人再按头让你按时起床了，没有那么多条条框框需要遵守了，但你在自由的空气中变成了一个六氟化铀气球，现实的引力不断拖拽着你向地面砸去。你不得不面对这个从来没有解答过的问题：是啊，做什么？

To be honest，很遗憾，我也不知道。

曾经的我以为大学和高中是一样的，只需要按部就班去做题，去考试就好了。时间来到大三下学期，保研季开始了。我猛然发现——同龄人早已甩出我太远太远。无数耀眼的论文竞赛奖项照耀着一穷二白的我，我能做的只有把几个课设混出来的项目东拼西凑变成几个简历上微不足道而又trivial的项目，然后在各种系统中得到默拒或明拒的结果。一场场面试不断打击着我，面试老师的质疑给我强有力的一记耳光：你改悔罢！

侥幸被目标院校录取后，我沉沦了半年。这期间我似乎被无形的大手上了一层强力的debuff，每天不是躺在床上睡大觉，就是打游戏。我去看了医生，医生告诉我，这是焦虑症。

后来我开始复盘，挖空了大脑里面几乎所有记得的过往。结论是，我并不是懒惰，而是走错了方向。

大学确实没有条条框框，但是另一层隐形的限制变得更明显了。别人已经了解、熟悉的东西，你可能听都没听过。“不知道做什么”，是很自然的问题。

一路上我碰了很多钉子。大一了解到ACM，却被蓝桥杯的题库爆杀，铩羽而归；了解到CTF，却连首页的东西都看不懂。大二稀里糊涂跟别人报了大创，最终无疾而终。虽然如此，最后还是侥幸取得了比较好的一个结果。

在即将毕业之际，回想起自己之前的四年，我有太多感慨。一方面庆幸自己还能继续学业，苟延残喘；另一方面，四年的时间让我认清了优绩主义的陷阱，弥补了我之前的一些遗憾。我希望自己能为和自己一样的人提供一些微不足道的帮助。

这个仓库就是为后来者设立的。收集这些学习资料、政策文件，不是让你继续像高中一样刷题的，而是希望你能少在无意义的事上做无用功，找到真正值得努力的方向。

有人说运气也是实力的一部分，对我来说可能确实如此。小学上了当时能选择的最好的初中，中考差5分侥幸上了目标高中，高考压着投档线上了吉大，保研又侥幸来到南大，冥冥之中似乎确有天意。但我想说的是，运气也是努力在时间上的积分。希望未来的你能在自己真正感兴趣的领域发光发热，尽早脱离做题的书山题海。
