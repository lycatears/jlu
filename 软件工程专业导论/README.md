# 软件工程专业导论
一门非常枯燥，但你又不得不赤的课。

## **警告：可能会有当堂检测，相当于纸质签到。**

1.5学分，课程性质为限选，直到25年9月22级推免工作开始后，限选课都不计入保研绩点（至少软院是这样的）。奖学金和转专业尚不明确，蓝色成绩单（或者叫校务行成绩单）上面是全科目成绩，会影响这个成绩，不过除了出国留学之外，蓝色成绩单的成绩似乎并没有什么用。

内容并不难，但是考试想拿到高分并不容易，据说24级删除了该课程的考试，可喜可贺，喜大普奔。
