## Python程序设计课设
机器学习项目的升级版。

### 使用方法
1. 确定conda虚拟环境，并安装依赖包
2. 运行后端目录下的sql文件创建数据库
3. 运行后端目录下的`startup.bat`启动后端服务
4. 在前端目录下运行`npm install`安装依赖包，然后运行`npm run dev`启动前端服务
5. 访问`http://localhost:5173/`
6. 注册一个账号，然后就能看到API key和演示界面
7. 在演示界面可以自行输入测试数据，也可以点击“示例”演示测试数据
