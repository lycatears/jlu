# 资料信息

- 朱允刚老师2023版PPT：个人认为价值非常高的一套讲义，包含部分往年真题、大厂面试题，还有很多扩展知识点，在考试内容之外仍有极高价值。如有最新版朱老师讲义，建议参照复习。

- 普班PPT：其他班级用的数据结构讲义，没有代码，比较抽象，不建议用这套讲义复习。

- 习题+答案：学习通作业题和答案。里面提供的真题很老，不建议使用这套真题复习。作业题基本都是万年不变祖传作业，答案可以参考一下。

- 408真题：数据结构考试有很多题都是408考研题改编，建议参考小题风格针对性练习。`算法题.pdf`是大题，近年来数据结构考试的大题也和408算法题比较相似。

- 其他资料：部分参考资料，包括课本、教辅、笔记等。

- 近几年的真题我没有找到，欢迎大佬补充。

- 建议有时间可以去听朱老师的课（学在吉大有录播），或者看王道的考研课，王道的408数据结构教材。
