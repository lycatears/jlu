# 操作系统原理【软件学院】
第四学期必修课，3学分，计入保研绩点。

虽然是一门专业课、一门必修课，但实际上和水课差不多。

**注意这里的课程是软件学院的，不是计院的，计院的考试难得多。**

## 考核
- 平时10，实验30，考试70。
- 不怎么签到，平时分主要靠学习通作业。学习通作业我们只留了一次，而且还是期末考试前留的。该作业似乎不会认真给分，态度好一点就行。
- 实验是修改几个代码，跑通。必须指出的是，该课程的实验指导PPT比我年龄大。实验报告最好多水一点，我水了51页，拿到了95分。
- 考试也很水，计算题甚至每年都是固定的，只不过改几个数据而已。全卷唯一的难点在于最后的PV操作&管程大题，26分，其他题目都是背书。

## 22级试卷结构
- 名词解释，5*2=10分。无英文题
- 单选题，10*1=10分。基本都是基础概念题。
- 简答题，6*5=30分。
- 计算题，4*6=24分。
- 管程与PV操作，2*13=26分。

## 关于计算题
- 每年的题型都是固定的，只不过改几个数据。
  - 内存页面置换
  - 银行家算法
  - 进程调度
  - 磁盘磁头调度
- 如何得到一套软件学院真题？去奔腾买一套操作系统题，选择题特别多的是计算机学院的，选择题很少甚至没有的是软件学院的，那套应该是2012年的真题。恭喜你得到了24分。
