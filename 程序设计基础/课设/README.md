# 程序设计基础课程设计：酒水管理系统
1.5学分，必修课，考核严格。

C语言程序设计的课设项目，按照课程要求采用链表+文件实现。本课程考核较为严格，五级制打分，普班基本上只有队长能拿到满绩，卓班情况尚不明确。答辩在10分钟以上，老师可能会亲自上手操作，需要注意程序的健壮性，比如处理非法输入。
## 解析
- `config.dat`：系统配置文件，是二进制文件，删除后在下次打开时会恢复到系统默认配置。
- `defines.h`：一些宏定义，以及系统配置文件的结构体定义。
- `good.c` & `good.h`：商品管理相关代码。
- `good.txt`：商品信息文件。
- `main.c`：程序入口，包括菜单，系统初始化等代码。
- `new.sln`：Visual Studio 项目文件
- `public.c` & `public.h`：一些高频使用的函数，例如安全输入输出，输入密码等。
- `record.c` & `record.h`：销售记录相关代码。
- `record.dat`：销售记录数据文件，二进制文件。
- `server_time.c` & `server_time.h`：时间接口相关封装。
- `suggest.txt`：意见箱记录文件。
- `user.txt`：用户信息文件。
- `userman.c` & `userman.h`：用户管理相关代码。
## 注意事项
- 默认管理员账号和密码都是`admin`
- 零售商品需要这样添加：商品管理-拆分商品
- **课设的成绩不只由你的项目决定**，还需要做学习通上面的选择题，听说25级又加了OJ考试
