# C语言作业代码
**梦开始的地方！**

收集了本人第一学期在学习通上提交的所有作业代码，全部为满分通过，全部为本人手写调试。

对于初学者而言，您不应直接复制粘贴这里的代码提交到学习通评测系统上，也不应该使用AI完成这些作业。如果您完全没有接触过编程，该课程的习题是较好的入门指导。仅当您遇到比较棘手的问题时，才应该参考他人代码或AI的意见。
# 格式
所有代码均采用`[作业编号]-[题号].c`的格式命名，题干在代码注释中体现。例如，作业0的第一题代码，文件名为`0-1.c`。

# 关于中国大学MOOC
22级的平时分是学习通20分+中国大学MOOC20分，也就是说有两个网课平台需要做。23级之后似乎不需要做中国大学MOOC了，学习通平时分变成30分。出于存档、参考等需要，这里也提供了相关代码，单独存放在`中国大学MOOC`目录下，但是平时作业的代码在平台上已经找不到了，只有考试的题目能看到，因此只有一小部分代码。
