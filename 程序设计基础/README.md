# 程序设计基础
你的第一门专业课，梦开始的地方！

必修课，3.5学分，计入保研绩点。考核较为严格。
## 考核
- 22级：学习通慕课20%，中国大学MOOC20%，上机实验40%，期末考试20%。没有笔试考试。
- 23级及之后：学习通慕课30%，上机实验40%，期末考试30%。没有笔试考试。
- 上机实验和期末考试的本质是一样的，都是OJ考试，排名给分。
- 22级具体考核标准：5分一档。
  - A级：每次考试的前5%是100分，5%-10%是95，10%-15%是90。
  - B级：15%-30%：85分，30%-50%：80分
  - C级：50%-65%：75分，65%-80%：70分
  - D级：80%-85%：65分，85%-90%：60分
  - E级：3场上机实验不全为E，实验成绩为50分；全为E但考试分数不为0，得40分；全为E且考试分数为0，得0分。
  - 3次正式实验课考试的排名会加到一起，得出的这个数字再排名，得到你的最终实验成绩；期末考试直接按照上述标准得分。
- 慕课注意不要随便提交章节测试，改不了，会扣分；注意完成讨论贴作业，在课程讨论区完成发帖任务。
- 上机考试的排名由你的成绩和用时共同决定，每次错误提交会有罚时20分钟。这个上机考试压迫感比较强，教室前面的大屏幕会有实时排名。
## 建议
- 不要觉得什么东西很可怕，比如很多教材把指针妖魔化，总是说他如何如何危险。我才不怕指针，无非就是越界访问，后面忘了。
- 慕课一定要记得刷。
