# 离散数学
一门难度较高的课程，计算机相关专业专享数学课。

- 软件学院：离散数学1：必修，3.5学分，计入保研绩点。离散数学2：必修，2学分，计入保研绩点，内容较少。
- 计算机学院：离散数学1：必修，3.5学分，计入保研绩点。离散数学2：必修，3.5学分，计入保研绩点，内容多，较为困难。
- 离散数学1：集合论，数理逻辑，图论，数论。
  - 软件学院2022级成绩分布数据：1-5班优秀5人，良好67人，中等48人，及格29人，不及格12人。最高分96，最低分34，平均分75。全课程优秀30人，良好158人，中等123人，及格120人，不及格35人。最高分99，最低分0，平均分73。
- 离散数学2：抽象代数（又称近世代数）。
  - 软件学院2022级成绩分布数据：1-5班优秀36人，良好57人，中等35人，及格26人，不及格10人。最高分98，最低分29，平均分79。全课程优秀99人，良好152人，中等91人，及格83人，不及格28人。
