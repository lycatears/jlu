# 新生研讨课
以PPT为核心的课程。

限选课，0.5学分，不计入保研绩点，不挂人。

## 流程
- 选题，由一个叫做“班主任”的老师发布在线表格，选择一个题目制作主题PPT。（实际上这个班主任似乎并没有发挥什么作用，除了上这个课以外）
- 新生研讨课分为两天，每次占用4节课时间。第一节课老师介绍一些学院的相关情况，询问一些成绩相关的问题。以行政班为单位上课。
- 第二节课需要提交：大学个人规划一份，PPT一份。
- 讲PPT，五分钟。
- PPT做的好看点就能拿优秀。
