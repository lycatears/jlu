## 面试抽题工具
- 一个基于wpf的程序，实现了本地题库+AI出题两种抽题模式。
- 需要自行配置AI接口、提示词，可以在程序菜单里面配置，也可以改`ProblemShow/config.json`，二者本质上是一样的。
- 默认本地题库是`ProblemShow/problemLibrary.json`
- 项目文件是`wpfApp1.sln`，用vs打开
