## 使用方法
1. 安装MySQL，使用`qq_users.sql`配置数据库
2. 自己填充一些数据
3. 项目文件是`Chat_test`目录下的sln文件，用VS打开，修改`db.cpp`里面的数据库用户名、密码、端口号
4. 编译客户端和服务端两个项目
5. 先启动服务端，再启动客户端，客户端可以多开几个，IP当然是我们万能的`127.0.0.1`
6. 在客户端登陆账号，根据程序菜单指导，进行聊天
