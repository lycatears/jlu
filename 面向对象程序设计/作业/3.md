试读以下代码片段，分析代码并讨论，代码是否有误，陈述原因。
```cpp
float t;

float f1(float r)

{  return(3.14*r*r);}

float &f2(float r)

{  return(3.14*r*r);}

float &f3(float r)

{  float f; f=3.14*r*r; return(f);}

float &f4(float r)

{  t=3.14*r*r; return(t);}



void main(){

   float a=f1(5); float & b=f1(5);

   float c=f2(5); float & d=f2(5);

   float e=f3(5); float & f=f3(5); 

   float g=f4(5); float & h=f4(5); 

}
```


我的答案：80分
在main函数中，首先声明了浮点型变量a，并调用f1进行赋值，f1返回浮点型数值78.5，赋值给变量a。

之后声明浮点型引用b，但是因为f1的返回值在函数运行结束后被释放，引用b指向了无效的内存地址，因此引起了编译错误。

函数f2的返回值是一个float类型的引用，但是没有指向的变量，因此c，d两个变量都不能成功赋值。

e的赋值来源于临时变量f，能够成功赋值，而f指向了被释放的栈区空间，引发错误。

g的值来自于全局变量t，能够成功赋值，h指向全局变量，也可成功赋值。

正确答案：
参考：f2定义编译通不过，引用需要内存地址作为依托，体现别名的作用；float & b=f1(5);同理编译通不过；float & f=f3(5);虽然有内存地址，但是根据函数工作机理，引用的地址依托为临时变量，函数工作完成就释放了，所以也得不到期望的答案。
