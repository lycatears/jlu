讨论：
以咱们面向对象课程授课为情景，试着讨论该情景下涉及的实体有哪些，对象有哪些，对象都需要哪些状态和行为，并试着定义抽象数据类型等等。
我的答案：90分
- 涉及的实体：学生，老师
- 对象：每一个学生，老师
- 学生的状态：姓名，学号，年龄，性别，专业，上课状态（出勤，缺勤，病假等）
- 学生的行为：听课，提问，回答老师的问题，记笔记
- 老师的状态：姓名，工号，年龄，性别，学位，课题
- 老师的行为：上课，签到，布置作业，提问，回答学生提问
- 学生的抽象数据类型：
```cpp
class student
{
    protected:
        string name;//姓名
        string major;//专业
        int id;//学号
        int age;//年龄
        int sex;//性别
        int status;//上课状态
    public:
        void ShowName();//显示姓名
        void Listen();//听课
        void AskQuestion();//提问
        void Answer();//回答
        void Write();//笔记
}
```
- 老师的抽象数据类型：
```cpp
class teacher
{
protected:
    string name;//姓名
    string degree;//学位
    string issue;//课题
    int id;//工号
    int age;//年龄
    int sex;//性别
public:
    void ShowName();//介绍自己
    void Teach();//讲课
    void AskQuestion();//提问
    void Answer();//回答学生提问
    void Check();//签到
    void HomeWork();//布置作业
}
```
